# Interest-Calculator
An interest calculator quickly computes interest on a principal amount using rate and time, helping users estimate returns or loan costs accurately.
Features
Calculate Simple Interest
Calculate Compound Interest
Supports Monthly and Yearly compounding
Displays:
Net Interest
Total Amount Payable
Clean UI with responsive inputs
Easy to use and lightweight

Technologies Used
HTML – Structure
CSS – Styling
JavaScript – Logic for interest calculation

How to Use
Open the index.html file in a browser.
Enter:
Principal Amount
Interest Rate (%)
Time Period (Years)
Select:
Interest Type (Simple / Compound)
Frequency (Monthly / Yearly)
Click Calculate.
View:
Net Interest
Total Payable Amount
